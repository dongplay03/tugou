# Frontend Notes

This frontend is part of the full TuGou Catcher project.

For complete setup, backend startup, frontend startup, API usage, and troubleshooting, see the root documentation:

- `../README.md`

Quick start for frontend only:

```bash
cd /Users/dongmac/Desktop/tugoucatcher/frontend
npm install
npm run dev
```

The frontend backend addresses are configured through `VITE_API_URL` and `VITE_WS_URL`.

If you want the full product workflow, use the repository root command:

```bash
cd /Users/dongmac/Desktop/tugoucatcher
npm install
npm run install:all
npm run dev
```
